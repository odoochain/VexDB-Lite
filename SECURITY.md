# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest (v0.0.x) | ✅ |
| Older releases | ❌ |

## Reporting a Vulnerability

Please **do not** report security vulnerabilities through public GitHub Issues.

Instead, report them via GitHub's private vulnerability reporting:
👉 [Report a vulnerability](https://github.com/VexDB-THU/VexDB-Lite/security/advisories/new)

Or email the maintainers directly (contact info in the repository).

### What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (optional)

We will work with you to assess and release a fix.
